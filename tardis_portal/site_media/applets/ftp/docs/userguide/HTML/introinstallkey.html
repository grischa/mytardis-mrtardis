Installing your license key

    

When you purchase a license key from JSCAPE or an authorized distributor, you will receive a confirmation e-mail containing your license key.  This license key is named license-key.txt  Upload the supplied license-key.txt file in the same directory on your server containing the applet jar file.

 

At this point close all instances of your browser and reload the applet.  The applet will then verify the host address against the license-key.txt file and the evaluation banner will be removed.

 

Note

 

If you are using a FTP client to perform this upload ensure that you are using binary mode when transferring the license-key.txt file, otherwise the key may become corrupted due to line conversions performed by FTP clients when in ASCII mode.
