Managing local and remote files

    

Both local and remote files may be managed using the context-sensitive pop-up dialogs found in the local and remote directory listings.  This pop-up dialog may be accessed by performing a right-click on the local or remote directory listing.  Available commands include.

 

Mkdir - Make a directory.

Rename - Rename a file or directory.

Delete - Delete a file or directory.

Refresh - Refresh a directory listing.

Resume - Resume a broken file transfer.
