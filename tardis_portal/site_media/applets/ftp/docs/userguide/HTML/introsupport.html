Technical support

    

At JSCAPE we strive to provide useful and timely technical support. If you have any technical support issues regarding Secure FTP Applet please post support questions to the Help Desk at http://www.jscape.com/support/

 

When requesting support please be sure to include any source code which may help us in answering your question. JSCAPE will make every effort to answer your technical support questions in a timely manner. Most technical support questions can be resolved within 12-24 hours.
